==============================================
  Combos HTML encadenados con jQuery Cascade
                     http://www.dosideas.com
==============================================

Este proyecto contiene un pequeño ejemplo sobre el uso de la librería JavaScript
jQuery junto al plugin jQuery Cascade, que permiten encadenar combos HTML de
manera simple

 Contenido
===========
 jquery/              : directorio con las librerías jQuery y jQuery Cascade
 demo.html            : página con un formuario HTML que utiliza jQuery Cascade
 datos-provincias.js  : archivo con los datos de las provincias
 datos-ciudades.js    : archivo con los datos de las ciudades


 Ejecución
===========
 Abrir el archivo "demo.html" con un navegador, y disfrutar :)
